# 快速啟動指南

## 簡介

這是一個加密貨幣投資分析網頁應用，展示前20大加密貨幣2020-2024年的投報率，並提供定期定額投資(DCA)計算器。

## 📋 需求

- Python 3.6+ 或任何支持的Web服務器
- 現代web瀏覽器 (Chrome, Firefox, Safari, Edge)
- 網際網路連接（用於載入Chart.js庫）

## 🚀 啟動方式

### 方式1: Windows 批處理文件（最簡單）

雙擊 `run_demo.bat` 文件：
```
run_demo.bat
```

這會自動：
1. 檢查Python安裝
2. 檢查數據文件
3. 啟動本地服務器
4. 打開網頁

### 方式2: Python 腳本

在命令行運行：
```bash
python run_demo.py
```

選項：
- 自動打開默認瀏覽器
- 顯示服務器狀態
- 容易停止和重啟

### 方式3: 直接 HTTP 服務器

在項目目錄打開命令行，運行：
```bash
python -m http.server 8000
```

然後在瀏覽器訪問：
```
http://localhost:8000
```

### 方式4: PowerShell（Windows 10+）

右鍵點擊 `run_demo.ps1`，選擇"使用PowerShell執行"

或在PowerShell中運行：
```powershell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
.\run_demo.ps1
```

## 📈 理解投報率數據

| 年份 | 市場情況 | 典型表現 | 特點 |
|------|--------|--------|------|
| 2020 | 牛市 | +200% 到 +12,000% | 強勁回升，大多數幣漲幅驚人 |
| 2021 | 延續牛市 | +50% 到 +15,000% | 極端波動，部分Altcoin漲幅超過100倍 |
| 2022 | 熊市 | -50% 到 -86% | 全面下跌，BTC, ETH 也下跌超60% |
| 2023 | 反彈年 | -30% 到 +150% | 分化明顯，部分幣反彈，部分繼續下跌 |
| 2024 | 強勁年 | +50% 到 +500%+ | 全面上升，BTC 創新高 |

### 閱讀數據的方式

- **正數%**: 該年投資收益（如+150%表示投資翻倍還多50%）
- **負數%**: 該年投資虧損（如-64%表示虧損64%）
- **5年投報率**: 5年累計回報（不是年均回報）

## 💡 DCA 計算器工作原理

### 定期定額投資 (DCA) 是什麼？

DCA = Dollar Cost Averaging 的縮寫。策略是：
- **固定時間**（例如每月）
- **固定金額**（例如₹500）
- **持續投資**（例如5年）

### 計算方法

該計算器基於歷史投報率模擬投資結果：

```
第1月: 投資初始 $1000
第2月: 前月資產 × (1+月度投報率) + $500新投資
第3月: 第2月資產 × (1+月度投報率) + $500新投資
...
第60月: 計算最終資產
```

### 例子

選擇：
- 幣種: Bitcoin (BTC)
- 初始投資: $1000
- 月度投資: $100
- 時間: 4年 (2020-2024)
- 起始年: 2020

結果（模擬）:
- 總投入: $5800 ($1000 + $100×48個月)
- 最終資產: ~$65,000-80,000
- 投報率: ~1000-1300%
- 說明: 通過分散投資，降低了市場時機風險

## ⚠️ 重要提示

### 免責聲明

1. **不是投資建議** - 本應用僅供教育用途
2. **過去不代表未來** - 歷史投報率不保證將來表現
3. **高風險** - 加密貨幣極不穩定
4. **模擬結果不準確** - 實際投資會受到多種因素影響
5. **請做足功課** - 投資前請充分研究

### 影響實際投資結果的因素

- **進場時機**: 同樣金額，進場時間不同結果差異巨大
- **市場波動**: 無法預測的短期波動
- **交易費用**: 實際交易會產生手續費
- **稅務**: 不同國家稅務政策不同
- **風險**: 加密貨幣市場存在極端風險

## 🔧 故障排除

### 1. "找不到最新版本 Python"
**解決方案：**
- 確保Python已安裝並在環境變量中
- 打開CMD，輸入 `python --version` 檢查
- 如未安裝，從 python.org 下載

### 2. "Cannot get crypto_data.csv"
**解決方案：**
- 檢查文件是否在項目目錄
- 確保使用HTTP服務器而非 file:// 協議
- 檢查文件權限

### 3. "圖表沒有顯示"
**解決方案：**
- 檢查網際網路連接（需載入Chart.js）
- 清除瀏覽器緩存
- 嘗試使用不同的瀏覽器

### 4. "表格顯示但沒有數據"
**解決方案：**
- 檢查瀏覽器開發者工具 (F12) 中的錯誤
- 確保CSV文件格式正確
- 重新刷新頁面

### 5. "DCA計算沒有結果"
**解決方案：**
- 確保選擇了有效的加密貨幣
- 檢查輸入的金額是否為正數
- 確保選擇的年份範圍有效
